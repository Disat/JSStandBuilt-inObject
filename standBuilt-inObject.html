parseFloat()

The parseFloat() function parses an argument (converting it to a string first if needed) and returns a floating point number.

不仅仅只能parse字符串，parseFloat will parse non-string objects if they have a toString or valueOf method. The returned value is the same as if parseFloat had been called on the result of those methods.
